Version Compare

Compare two versions while keeping the legacy text intact and showing proposed edits as separate suggestions.

Help
  1. Select an input mode: Pasted Text, Files, or URLs.
  2. Provide original and updated versions.
  3. Click Compare to view summary plus a legacy-preserving redline.
  4. Export with Download HTML Report or Download PDF Report.

URL mode may be blocked by CORS/auth on private links. If that happens, paste text or upload files.

Recommended start: Pasted Text for the fastest compare.
